# Green Bridge

An eco-focused web platform built using PHP and MySQL, with a frontend powered by HTML, CSS, and JavaScript.

## Features
- User registration and login
- Session-based authentication
- User dashboard
- QR code generation
- Database integration with MySQL

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Files Overview
- index.php – landing page
- login.php / register.php – authentication
- dashboard.php – user dashboard
- profile.php – user profile
- generate_qr.php – QR code generation
- db.php – database connection

## Purpose
This project was built to practice backend development, authentication, and database-driven PHP applications.
